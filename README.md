## 前言

神三6.1是果子系统，没有能力重算，故采用压级炼果打法，尽量保证我军武将的完美成长。

经验的计算法则：

- 对同等级敌军进行物理攻击得6点经验，每高1级，经验+2，收同等级人头得38点经验，每高1级，经验+2。
- 对同等级我/友/敌军施法得10点经验，每高1级，经验+4，收同等级人头得32点经验，每高1级，经验+8。
- 群法、穿透一次收多个人头的话，经验只按等级最高的一个算。
- 扔手榴弹的经验同物理攻击。
- 地雷炸伤敌军不得经验，炸死敌军（地雷伤害>敌军剩余血量）全军得经验，经验计算同物理攻击击退敌军；若伤害恰好等于敌军剩余血量，则不得经验。

压级的主要思路：

- 强力友军：直接清场，如第二元神（主角带轩辕剑+引导马）、吕布、冉闵、高宠这些自带aoe的武将。
- 弱鸡友军：我军用众志成城辅助他们，有了火器后，可以让他们扔火器，敌军血厚的话，我军可先用地雷消耗敌军（虎牢关、天门阵）。
- 不可控弱鸡友军：多创造人头引导他们进鹿寨之类的优势地形，拉他们围攻（三辅）。
- 没有友军但有强力敌军：心控强力敌军清兵（广宗、青城山）。心控+14级敌军经验是10+4\*14=66=38+2\*14，正好等于人头经验，所以只要敌军不超过我军14级，最后过关时利用心控都比打死划算。
- 有褒姒：可用褒姒分裂敌军，再用众矢之的拉着某方的强力敌军清掉另一方，其余我军帮助弱的另一方即可（坂泉）。
- 没有狐妖、敌军也没有会aoe的：只能我军自己手打（梁山），此时一定注意出手效率，轻易不出手，出手就一定是多个人头，1级敌军优先用群法收，2级敌军两可，3级及以上优先用穿透收。

放弃的单挑：

- R场景的所有手动单挑，都吃道具混过去。
- S战场加血：地府（孙悟空vs黄天化，+50）、万妖夜行（孙悟空vs刑天，+50）、逐鹿1（八戒vs庞统，+100），血多只会增加出特技的难度。
- S战场加天赋：陈仓（赵云vs阎行，破坏防御）、梁山（岳飞vs林冲，自动提升爆发）、梁山（张飞vs张顺，随机状态攻击）、天门阵（婴宁vs妲己，随机属性攻击）、逐鹿1（白素贞vs法海，无反攻击），不打人不需要攻击型天赋。
- S战场得装备：梁山（武松vs鲁智深，水磨禅杖）、赤壁（李存孝vs罗宣，火尖枪）、天门阵（孙悟空vs杨戬，三尖两刃刀），不打人也不需要武器。
- 天门阵（主角vs法海），不放弃要多打两拨兵，回合不够。

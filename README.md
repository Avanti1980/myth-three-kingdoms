## 前言

神三6.1是果子系统，没有能力重算，故采用压级炼果打法，尽量保证我军武将的完美成长。

经验的计算法则
1. 物理攻击同等级敌军得6点经验，敌军每高1级，经验+2，收同等级人头得38点经验，敌军每高1级，经验+2
2. 法术攻击同等级敌军得10点经验，敌军每高1级，经验+4，收同等级人头得32点经验，敌军每高1级，经验+8
3. 群法、穿刺一次收多个人头的话，经验只按等级最高的一个算
狐妖心控+4级的敌军，经验是10+4*4=26，相比自己直接得人头经验，心控还是划算的
所以杀敌的优先级就很简单了
1. 强力友军直接一人清全场，如吕布、冉闵这些中途会短暂加入的武将
2. 弱鸡友军，我军用众志成城辅助他们，有了火器后，可以让他们扔火器，敌军血厚的话，我军可先用地雷消耗敌军（虎牢关）
3. 不可控弱鸡友军，多创造人头引导他们进鹿寨之类的优势地形，拉他们围攻（三辅）
4. 没有友军，但有如吕布这些会aoe的强力敌军，用狐妖心控（广宗）
5. 没有狐妖或者敌军也没有会aoe的，只能我军自己动手（梁山），此时一定注意出手效率，轻易不出手，出手就一定是多个人头，1级敌军优先用群法收，32 vs. 38，2级敌军随意，都是40，其余优先用穿刺收

经验的计算规则
1. 对同等级敌军进行物理攻击得6点经验，每高1级，经验+2，收同等级人头得38点经验，每高1级，经验+2
2. 对同等级我/友/敌军施法得10点经验，每高1级，经验+4，收同等级人头得32点经验，每高1级，经验+8
3. 群法、穿透一次收多个人头的话，经验只按等级最高的一个算
心控+7级的敌军，经验是10+4*7=38，正好等于人头经验，所以只要敌军不超过我军7级，心控都是划算的，所以杀敌的优先级为
1. 强力友军直接一人清全场，如第二元神、吕布、冉闵、高宠这些自带aoe的武将
2. 弱鸡友军，我军用众志成城辅助他们，有了火器后，可以让他们扔火器，敌军血厚的话，我军可先用地雷消耗敌军（虎牢关、天门阵）
3. 不可控弱鸡友军，多创造人头引导他们进鹿寨之类的优势地形，拉他们围攻（三辅）
4. 没有友军，但有如吕布这些会aoe的强力敌军，用狐妖心控（广宗）
5. 如果有褒姒，可用褒姒分裂敌军，再用八戒拉着会aoe的强力敌军清掉某一方，其余我军帮助弱的另一方即可（坂泉八戒拉蚩尤）
6. 没有狐妖或者敌军也没有会aoe的，只能我军自己手打（梁山），此时一定注意出手效率，轻易不出手，出手就一定是多个人头，1级敌军优先用群法收，2级敌军两可，3级及以上优先用穿透收

单挑的取舍：R场景的单挑都吃道具混过去，S战场加血的单挑都放弃，血多只会耽误变身，天门阵婴宁（随机属性攻击）、逐鹿一白素贞（无反攻击）加天赋的单挑放弃，不打人也就不需要攻击型天赋了，天门阵主角的单挑放弃，否则要多打两拨兵，回合不够
